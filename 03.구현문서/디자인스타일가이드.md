## 2. 글로벌 레이아웃

### 2.1 레이아웃 구조

```
+-----------------------------------------------------------+
|                    Header (64px)                           |
+------------+----------------------------------------------+
|            |                                              |
|  Sidebar   |              Main Content                    |
|  (260px)   |              (fluid, max 1440px)             |
|            |                                              |
|            |                                              |
+------------+----------------------------------------------+
```

| 영역 | 크기 | 배경색 | 비고 |
|------|------|--------|------|
| 헤더 | H: 64px, 전체 너비 | [색상] | 검색, 알림, 프로필 |
| 사이드바 (확장) | W: 260px | [색상] | 내비게이션 |
| 사이드바 (축소) | W: 72px | [색상] | 아이콘 전용 |
| 콘텐츠 영역 | Max-W: 1440px | [색상] | 32px 패딩 |

### 2.2 그리드 시스템

| 항목 | 값 |
|------|-----|
| 컬럼 수 | 12 |
| 컬럼 간격 | 24px |

| 콘텐츠 최대 너비 | 1440px |

## 6. 핵심 UI 컴포넌트

### 6.1 버튼

| Variant | 배경 | 텍스트 | 보더 | 용도 |
|---------|------|--------|------|------|
| Primary | [색상] | [색상] | 없음 | 주요 액션 |
| Secondary | 투명 | [색상] | 1px [색상] | 보조 액션 |
| Danger | [색상] | White | 없음 | 삭제, 위험 액션 |
| Ghost | 투명 | [색상] | 없음 | 텍스트 링크형 |

**크기**: sm (32px), md (40px), lg (48px)
**Border-radius**: 8px
**상태**: Default, Hover, Active, Disabled, Loading

### 6.2 입력 필드

| 속성 | 값 |
|------|-----|
| 높이 | 40px |
| Border-radius | 12px |
| 기본 보더 | 1px [Gray 200] |
| 포커스 보더 | 1px [Primary] |
| 에러 보더 | 1px [Danger] |
| 라벨 | 위치: 상단, body-sm / SemiBold |

### 6.3 카드

| 속성 | 값 |
|------|-----|
| 배경 | White |
| Border-radius | 12px |
| 보더 | 1px rgba(0,0,0,0.06) |
| 패딩 | 24px |
| 호버 | translateY(-2px) + shadow |

### 6.4 배지 & 태그

| 유형 | 스타일 | 용도 |

| Count Badge | 원형, 숫자 | 알림 카운트 |
| Category Tag | 라운드 필, 라이트 배경 | 카테고리 분류 |

### 6.5 데이터 테이블

| 속성 | 값 |
|------|-----|
| 헤더 스타일 | overline / uppercase / Gray 400 |
| 셀 스타일 | body / Gray 700 |
| 행 호버 | Gray 50 배경 |
| 구분선 | 1px Gray 100 |
| 페이지네이션 | 하단, 10/20/50/100 선택 |

### 6.6 모달

## 9. 반응형 & 접근성

### 9.1 브레이크포인트

| 이름 | 크기 | 용도 |
|------|------|------|
| Desktop | 1440px+ | 기본 레이아웃 |
| Tablet | 1024px | 축소 사이드바 |
| Mobile | 768px | 비상 접근용 |

### 9.2 접근성 (WCAG 2.1 AA)

| 항목 | 기준 |
|------|------|
| 포커스 상태 | 2px outline, Primary 컬러, 2px offset |
| 텍스트 대비율 | 최소 4.5:1 (AA) |
| 터치 타겟 | 최소 44x44px |
| 상태 표시 | 색상 + 아이콘 (색상만 사용 금지) |
| 에러 표시 | 아이콘 + 텍스트 + 보더 (3중 코딩) |
| 모션 | prefers-reduced-motion 지원 |
