## 1. Executive Summary

### 1.1 한 줄 요약

**"오늘 점심, 어디서 먹지?"를 5초 안에 해결하는, 점심 시간대 × 위치 기반 × 일반 식당의 쿠폰형 마감할인 플랫폼.** GS25가 1년 만에 매출 134.5%를 만들어낸 '마감할인 잉여수익화 모델'의 빈 슬롯(점심 시간대 26.1%) × 일반 식당 진영을 선점한다.

### 1.2 핵심 지표

| 항목 | 내용 |
|------|------|
| 서비스명 | **점심특강 (Lunch Special Lecture)** |
| 목표 시장 | 국내 외식 시장 약 150조 원 + 푸드테크 인접 시장 200조 원 TAM (NLM-30) |
| 타겟 사용자 | 직장인(20~40대 사무직, Primary) / 주부(30~50대, Secondary) / 식당 사장님(Supplier) / B2B 법인 (Phase 2) |
| 핵심 가치 | **점심 시간대 1만 원 이하 가성비 식당을 5초 안에 발견 + 제휴 쿠폰** |
| 수익 모델 | 광고(메인) + 쿠폰 사용 수수료 + 사장님 프리미엄 멤버십 + B2B 법인 식권 (Phase 2) |
| 목표 기간 | **6개월** (MVP 5개월 + 안정화 1개월) |
| 6개월 KPI | MAU 50,000 / 등록 매장 3,000 / 쿠폰 사용률 40% / 월 광고 매출 5,000만 원 |
| 예산 | 3억 원 (인건비 70% 포함) |

### 2.2 시장 기회

- **기회 1 — 점심 시간대 × 일반 식당 마감할인의 빈 슬롯**: GS25 마감할인 시간대 분포(저녁 55.8% vs 점심 26.1%)가 점심대 미개척을 직접 증명. 럭키밀(베이커리), 라스트오더(저녁), 편의점(저녁 중심)이 각자 다른 영역을 차지한 가운데 **점심 시간대 × 일반 식당** 영역은 메이저 부재. — 근거: 마켓리서치 #2-6, #2-10, #2-11
- **기회 2 — 잉여수익화 모델의 한국 시장 검증 완료**: GS25 마감할인이 134.5% 신장, 80만 개+ 판매로 메커니즘이 정량적으로 입증됨. 동일 구조를 일반 식당 점심특선에 이식. — 근거: 마켓리서치 #2-6
- **기회 3 — 런치플레이션 YoY 가속으로 사용자 가입 동기 강화**: 비용 부담 65.8→67.5%, 간편식 의존 47.0→54.1%로 모두 상승. "5초·1만 원 이하" 가입 트리거. — 근거: 마켓리서치 #2-1, #2-3
- **기회 4 — 사장님의 신규 유입 채널 갈증**: 외식 사업체 1만 1,000개 순감 + 한식·동네식당 객수 감소. 점심 비수기 좌석 1개 = 새 수익 1단위. — 근거: 마켓리서치 #2-5, #2-7
- **기회 5 — B2B 법인 멤버십의 정량 근거 확보**: **임직원 84.6%가 식비 지원형 복지를 선호**(NLM-14). 구내식당 미보유 중소기업 공략을 위한 Phase 2 핵심 자산. — 근거: 마켓리서치 #2-2

### 2.3 경쟁 환경

| 경쟁사 | 강점 | 약점 | 시장 점유율 |
|--------|------|------|-------------|
| **럭키밀** | 시간 일치성(마감=퇴근), 50% 고정 할인, 베이커리 큐레이션 | 베이커리 한정, 서울 중심, 픽업 의존 | 10만+ 사용자 (출시 1년) |
| **라스트오더 (미로)** | 다양한 카테고리, 평균 40% 할인, 윈-윈-윈 모델 | **저녁/야간 중심, 점심 비중 낮음** | 미공개 |
| **마감히어로** | 찜·선착순 기능, 임팩트스퀘어 투자 | 신생, 사용자 규모 작음 | 초기 단계 |
| **GS25 마감할인** | **2025년 매출 +134.5%, 누적 80만 개**, 1위=도시락, 전국 점포망 | **시간대 저녁 55.8% 편중, 점심대 26.1%, 일반 식당 미포함** | 전국 수만 점포 |
| **편의점 PB 도시락** (CU/세븐일레븐 포함) | 위치·가격·속도 3박자, 4~5천 원대 (50% 할인 시 2,000원대) | 일반 식당 경험 부재 | 전국 수만 점포 |
| **Too Good To Go** (글로벌) | 글로벌 1위, 환경 가치 결합, 50~70% 할인 | **국내 미진출** (잠재 위협) | 글로벌 7,000만+ |
| **카카오맵·네이버지도** | 압도적 데이터·사용자 기반 | 할인·쿠폰 부재, 사장님 자가 운영 도구 미흡 | 전 국민 |
| **점심특강 (본 프로젝트)** | **점심 특화 + 일반 식당 + 쿠폰·통계 결합 (빈 슬롯 단독)** | 신규 진입, 매장 등록 부진 리스크 | **진입 예정 (0 → MAU 50K, 6M 목표)** |

**포지셔닝 한 줄**: *"라스트오더가 저녁의 잉여를 풀었다면, 점심특강은 점심의 빈자리를 채운다. GS25가 키운 마감할인 시장의 점심 26.1% 슬롯 + 일반 식당 진영을 가져온다."*

## 3. 타겟 사용자 정의

### 3.1 사용자 페르소나

**페르소나 1: 김대리 (직장인 · Primary)**

| 항목 | 내용 |
|------|------|
| 연령/직업 | 29세, 강남 사무지구 IT 회사 대리 (사무직) |
| 핵심 니즈 | 점심시간 1시간 내 회사 반경 500m 이내에서 8,000~12,000원 가성비 식당을 빠르게 결정 |
| 현재 해결 방식 | 카카오맵 검색 → 동료 추천 → 결정 실패 시 편의점 도시락 (5,000원) |
| 불만 사항 | 가격·할인 정보 불투명 / 혼자 가능한지 알 수 없음 / 대기 시간 예측 불가 / 관계 피로 (분위기 후식 33.1%, 메뉴 강요 28.5% YoY 상승) |
| 기대 가치 | "5초 안에 1만 원 이하 점심 결정 + 관계 부담 없는 1인 좌석 식당" |

**페르소나 2: 이미경 (주부 · Secondary)**

| 항목 | 내용 |
|------|------|
| 연령/직업 | 42세, 30대 자녀와 거주, 가족·모임 외식 결정권자 |
| 핵심 니즈 | 가족 단위 모임을 위한 합리적 가격대(인당 1만 원~1.5만 원) 동네 식당 발견 |
| 현재 해결 방식 | 맘카페·블로그 후기 검색 → 광고성 후기 다수, 신뢰 어려움 |
| 불만 사항 | 광고·실제 후기 구분 어려움 / 실시간 할인 정보 분산 |
| 기대 가치 | "신뢰할 수 있는 동네 맛집의 점심특선·할인 쿠폰" |

**페르소나 3: 박사장 (식당 사장님 · Supplier)**

| 항목 | 내용 |
|------|------|
| 연령/직업 | 50세, 강남 30석 한식당 운영, 직원 3명 |
| 핵심 니즈 | 점심 비수기(13:30~14:30) 빈 좌석 채우기, 신규 고객 유입, 단골화 |
| 현재 해결 방식 | 전단지, 배달앱 광고 (월 30~50만 원, ROI 측정 불가) |
| 불만 사항 | 인건비 부담(2026년 최저임금 10,320원, 외식업 76.3% 부담 호소) + 마케팅 ROI 불투명 + 노쇼(1회 평균 44만 원 손실) |
| 기대 가치 | "1주일에 30분 투자해서 점심 객수 N명 늘어나는 게 보이는 도구" |

**페르소나 4: HR 김과장 (B2B 법인 · Phase 2)**

| 항목 | 내용 |
|------|------|
| 연령/직업 | 38세, 강남 50인 규모 IT 스타트업 인사·총무팀 |
| 핵심 니즈 | 임직원 식비 복지 제공 (구내식당 신축은 불가, 식대 현금 지원은 관리 부담) |
| 현재 해결 방식 | 월 식대 10만 원 현금 지원 + 회식비 별도 |
| 불만 사항 | 임직원이 어디 갔는지 모름, 정산 영수증 관리 부담 |
| 기대 가치 | **"임직원 84.6%가 선호하는 식비 지원형 복지"를 모바일 식권 쿠폰으로 자동화** |

## 4. 서비스 전략

### 4.1 핵심 가치 제안 (Value Proposition)

| 대상 | 핵심 가치 | 정량 근거 |
|------|----------|----------|
| **직장인** | "점심시간 1시간 안에 회사 반경 500m, 1만 원 이하 점심특선을 5초로" | 점심값 부담 67.5%, 가격 우선 67.9%, 점심시간 1시간 이내 56.3% |
| **주부** | "가족 외식·모임 합리적 가격대 동네 맛집, 검증된 쿠폰까지" | 맘카페 광고성 후기 한계, 가성비 매장 11% 성장 |
| **식당 사장님** | "점심 비수기 빈 좌석을 자동으로 채우고, 객수·재방문이 숫자로 보이는 단 하나의 마케팅 도구" | GS25 마감할인 134.5% 신장 검증, 30일 재방문율 KPI 표준 |
| **B2B 법인 (Phase 2)** | "임직원 84.6%가 선호하는 모바일 식권 쿠폰, 구내식당 신축 없이 식비 복지 완성" | 식비 지원형 회사 선호 84.6%, 구내식당 정성 한계 |

### 4.3 차별화 전략

1. **점심 시간대 × 일반 식당 × 위치 기반 = 단독 영역 선점**: GS25 마감할인의 점심 슬롯(26.1%) + 일반 식당 진영을 단일 카테고리로 깊이 있게 공략. 럭키밀 약점(베이커리 한정)·라스트오더 약점(저녁 중심)·편의점 약점(일반 식당 부재)이 모두 본 서비스의 강점.
2. **시간 일치성 마케팅**: 럭키밀의 검증된 공식 "마감 시간 = 퇴근 시간"을 "점심특선 시간(11:00~14:00) = 직장인 점심시간"으로 정확히 번안. 위치 500m + 시간 윈도우 푸시 알림으로 즉시성 극대화.
3. **사장님 ROI 가시화 대시보드 (락인 무기)**: Razzoo's 점심특선 KPI 공식(30일 재방문율 + 쿠폰 사용 경로)을 한국형 4지표(신규/재방문/사용률/객단가)로 1화면 집약. 인건비·노쇼·점심 비수기 3대 통점을 동시 해결한다는 메시지.
4. **B2B 법인 멤버십 (Phase 2 차별화)**: "임직원 84.6%가 식비 지원형 복지를 선호" 정량 근거로 구내식당 미보유 중소기업을 공략. 경쟁사 대비 유일한 B2B 채널 확보.
5. **관계 피로 해소형 1인 점심 포지셔닝**: 20·30대 혼밥률 50%+ × 관계 스트레스 지표 YoY 상승을 정조준한 "1인 좌석 / 빠른 결제 / 즉시 픽업" 메타데이터 필터.
6. **"진짜 식당 경험 with 편의점 가격"**: 편의점 도시락 2,000원대 대체재로 5,000~9,000원 점심특선(할인가) 카드를 메인 피드 상단 배치.
