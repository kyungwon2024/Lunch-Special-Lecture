## 크리티컬 패스 (Critical Path)

다음 경로는 지연 시 전체 일정에 직접 영향을 줍니다.

```
W1 PRD 승인
   ↓
W3 기획 완료 (2.3.1 API스펙 → 모든 개발 선행)
   ↓
W4 DB 스키마 (3.2.2 → 모든 개발 선행)
   ↓
W6~W9 MVP1 — F4 사업자 인증 (4.1.2 → MVP1 완료 게이트)
   ↓
W10~W13 MVP2 — F5 통계 (4.3.6 → 사장님 가치 제안 핵심)
   ↓
W14~W17 MVP3 — F3 쿠폰 토큰 (4.4.1 → 보안·악용 방지 핵심)
   ↓
W18~W19 베타 모집 (5.3.1, 5.3.2 → 검증 가능성 결정)
   ↓
W21 출시 (6.1.1 마켓 등록)
```

**완충 (Buffer)**: 각 단계 사이 0.5~1주씩 잠재 버퍼 포함 (예: MVP1 4주 중 통합 테스트 1주). 추가 지연 시 Should/Could 기능 (리뷰·즐겨찾기) 후순위로 보호.
