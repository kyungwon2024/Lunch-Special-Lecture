## 도구

- **Claude Code**: AI 기반 산출물/코드 자동 생성
- **GitHub**: 프로젝트 관리, 이슈 트래킹, CI/CD
- **n8n**: 워크플로우 자동화 (GitHub-Slack 연동, AI Agent)
- **Slack**: 팀 커뮤니케이션, 알림, AI PM Agent
- **NotebookLM**: 프로젝트 위키 + 통합자료실, 산출물/참조자료 통합 조회, AI Q&A
- **Supabase**: PostgreSQL DB + Storage + REST API (권장)
- **Vercel**: 프론트엔드/API 배포 (권장)

## 시작하기

1. `.AP-key.md`에 서비스 URL과 인증 키를 기입합니다
2. `00.통합자료실/`에 프로젝트 참조 자료(RFP, 정책, 인프라 등)를 저장합니다
3. NotebookLM에 프로젝트 노트북을 생성하고, PRD.md + 통합자료실 자료를 소스로 등록합니다
4. Claude Code를 실행합니다: `claude`
5. 첫 명령을 입력합니다: `claude "PRD 기반으로 프로젝트 셋팅해줘"`
