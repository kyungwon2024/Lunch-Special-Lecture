## 1. 프로젝트 개요

### 1.1 사업 정의
점심특강은 직장인과 주부가 주변 식당의 **점심특선·할인 정보**를 한눈에 확인하고 **제휴 쿠폰**을 받을 수 있는 **위치 기반 맛집 앱**이다. 식당에는 **광고와 쿠폰 발행**을 통한 신규 고객 유입 채널을 제공하는 **양면 시장(2-sided market) 플랫폼**으로 정의한다.

### 1.2 추진 배경 (시장 근거)

#### ① 런치플레이션의 일상화 — 직장인 점심 부담 가중
- 서울 지역 외식 가격 1년 새 평균 4%대 상승 (칼국수 9,923원 +4.9%, 냉면 12,538원 +4.2%) — [NLM-19](../00.통합자료실/참고자료/NLM-19_고물가틀깬파격_초저가점심값_YTN.md)
- 직장인 10명 중 7명이 "점심값이 많이 올랐다"고 체감 — [NLM-27](../00.통합자료실/참고자료/NLM-27_직장인10명중7명_점심값많이올랐다_KSA.md)
- 가성비 메뉴로의 수요 이동 가속 — [NLM-23](../00.통합자료실/참고자료/NLM-23_런치플레이션_가성비메뉴몰리는직장인_매일일보.md), [NLM-14](../00.통합자료실/참고자료/NLM-14_런치플레이션_구내식당선호_트렌드모니터.md)
- 시장 검증 사례: 홈플러스 990원 도시락 **4만 개 완판** — [NLM-19](../00.통합자료실/참고자료/NLM-19_고물가틀깬파격_초저가점심값_YTN.md)

#### ② 소상공인 마케팅 채널의 비효율
- 전단지·블로그 광고는 **유입 측정 불가** + ROI 낮음 (PRD 1.3)
- 점심 피크 외 시간대 빈 좌석 문제 — 비수기 시간대 **좌석 회전율** 핵심 KPI 부재

#### ③ 마감할인·외식 O2O 시장 성장 (벤치마크)
- 라스트오더(국내 마감할인): 시범 운영 시 **주 거래액 10만→100만 원** 성장, 사용자 만족도 **4.6/5** — [NLM-18](../00.통합자료실/참고자료/NLM-18_음식점마감세일앱_인터뷰_비즈한국.md)

- Too Good To Go (글로벌): SDG 12.3 음식물 쓰레기 50% 감축 목표 부합 모델 — [NLM-16](../00.통합자료실/참고자료/NLM-16_TooGoodToGo_ExecutiveSummary.md), [NLM-17](../00.통합자료실/참고자료/NLM-17_TooGoodToGo_케이스스터디_FutureLearn.md)

## 3. 프로젝트 범위

### 3.1 범위 내 (In Scope)  ·  MVP

#### 소비자 앱 (F1~F3)
- **F1. 위치 기반 식당 탐색**: 반경(500m/1km/2km) 지도·리스트 뷰, 카테고리 필터
- **F2. 점심특선 한눈에 보기**: "오늘의 점심특선" 메인 피드, 가격·할인율 카드, 실시간 영업 상태
- **F3. 제휴 쿠폰**: 발급/사용/만료 관리, QR/바코드 제시, 사용 알림

#### 사장님 앱 (F4~F5)
- **F4. 매장·특선 관리**: 사업자 인증, 일별/요일별 특선 등록, 즉시/예약 발행
- **F5. 쿠폰 발행 + 통계**: %/원 쿠폰, 신규/단골 타게팅, 일/주/월 통계 대시보드

### 3.2 범위 외 (Out of Scope)
| 항목 | 사유 | 향후 검토 |
|------|------|----------|
| 배달 기능 | "방문 외식" 포지셔닝 유지 (PRD G) | 영구 제외 |
| 테이블 예약 시스템 | MVP 복잡도 관리 | Phase 4 (M13~) 검토 |
| 결제 모듈 자체 운영 | PG 연동 부담 | Phase 3 카카오·네이버페이 |
| 리뷰·평점 | 콘텐츠 부족 시 부정 효과 | Phase 2 (등록 매장 1,000 이후) |
| 광고 상품 정식 운영 | 충분한 사용자 풀 확보 후 | Phase 3 (M7~) |

### 3.3 가정과 제약
- **가정**: 서울 강남·여의도 오피스 식당 최소 500개 초기 등록 가능 (영업팀 6주 가동 전제)
- **제약 1**: 사장님의 모바일 활용도 차이 → 사용성 단순화 필수
- **제약 2**: 쿠폰 악용(양도/도용) 차단을 위한 본인 인증 의무화
- **제약 3**: 식당 노쇼 등 정책 이슈 사전 검토 — [NLM-25](../00.통합자료실/참고자료/NLM-25_식당노쇼_예약부도피해예방지원_토스플레이스.md)
